On this page you will find the python documentation for the project: “Open source weather station.”

Author: Maurice Markvoort - Graduate student at Xablu. Email: m.markvoort@xablu.com

A Pycom LoPy is used as a microcontroller to read data and send it to The Things Network via LoRaWAN.

The sensor BME280 is used to read temperature, pressure and humidity.
