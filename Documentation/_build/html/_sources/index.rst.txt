Home
====

On this page you will find the python documentation for the project: "Open source weather station."

Author: Maurice Markvoort - Graduate student at Xablu.
Email: m.markvoort@xablu.com

A `Pycom <https://www.pycom.io/>`_ LoPy is used as a microcontroller to read data and send it to `The Things Network <https://www.thethingsnetwork.org/>`_ via LoRaWAN.

The sensor `BME280 <https://www.adafruit.com/product/2652>`_ is used to read temperature, pressure and humidity.
